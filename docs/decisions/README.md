# Decisiones técnicas (ADRs livianos)

Cada decisión no trivial que tomamos durante el desarrollo se documenta como un archivo aparte: `NNN-titulo-corto.md`.

Formato sugerido (mantenelo corto):

```markdown
# 001 — Título corto

**Estado**: aceptada | reemplazada por NNN | obsoleta
**Fecha**: YYYY-MM-DD

## Contexto
¿Qué problema estábamos resolviendo?

## Decisión
¿Qué elegimos y por qué?

## Alternativas consideradas
¿Qué descartamos y por qué?

## Consecuencias
¿Qué cambia ahora? ¿Qué pagamos en el futuro?
```
